## About the Project

CodeShieldAI is a full-stack application security scanning platform that combines **fine-tuned Transformer models** (GraphCodeBERT / CodeBERT) with a **deterministic Multi-Language Heuristic Engine** to deliver high-confidence vulnerability detection across **8+ programming languages**.

### Key Highlights

- Hybrid AI + rule-based detection pipeline
- Supports file uploads and GitHub repository scanning
- Exportable PDF & JSON security reports
- Dashboard with analytics, trends, and model comparison
- Real-time remediation guidance with line-level precision
